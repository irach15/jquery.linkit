#LinkIt jQuery Plugin#

**LinkIt** is a simple jQuery plugin to attach links to any DOM elements

##Version##
0.2.0

##Usage##
    $('span').linkIt({
       href: 'https:/jquery.com',
       text: 'click me',
       target: '_blank'
    });

##Vendors##
jQuery - [http://jquery.com](http://jquery.com)
